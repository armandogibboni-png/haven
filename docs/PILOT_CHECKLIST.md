# Haven Pilot Checklist

*Complete all P1 items before the pilot launches. P2 items during. P3 post-pilot.*

---

## ✅ P1 — Before launch

### Technical (developer)
- [ ] Crisis escalation button tested (opens correct NCSE/CAMHS links)
- [ ] SMQ pre-administration completed for each enrolled student (T0)
- [ ] GAS levels defined for at least one goal per student
- [ ] SEN Plan started and follow-up timer initialised (`initFollowUp`)
- [ ] Fidelity badge visible in Feedback tab
- [ ] RE-AIM Readiness page accessible from Settings
- [ ] Anthropic DPA requested and on file

### School / operational
- [ ] Pilot MOU signed by principal and project lead
- [ ] Teacher(s) enrolled with PIN and onboarded (30 min session)
- [ ] SNA enrolled if applicable (read-only access confirmed)
- [ ] Parental consent obtained for each enrolled student
- [ ] Families enrolled: parent profile created for each participating family
- [ ] Students enrolled with SM / SEN flag set

### Documentation
- [ ] Teacher Quick-Start Guide shared with teacher
- [ ] Privacy Policy URL communicated to all users
- [ ] Emergency / escalation contacts confirmed with school

---

## 🔄 P2 — During pilot (weekly)

### Teacher
- [ ] ≥ 1 observation logged per student per week
- [ ] Goals reviewed and statuses updated as needed
- [ ] Any revision proposals submitted to parents

### Parent
- [ ] ≥ 1 home observation per week
- [ ] Pending revisions reviewed and responded to

### Project lead
- [ ] Fidelity badge checked weekly (target: green)
- [ ] Inactivity prompts monitored
- [ ] Week 5: mid-pilot check-in call with teacher

---

## 📋 P3 — End of pilot

- [ ] SMQ T1 administered for each student
- [ ] GAS scores reviewed against T0 baselines
- [ ] End-of-pilot survey completed by teacher(s)
- [ ] End-of-pilot survey completed by parent(s)
- [ ] RE-AIM Readiness page screenshot taken for report
- [ ] All SEN Plans exported (↓ button in each plan)
- [ ] Pilot report written (template: `docs/PILOT_REPORT_TEMPLATE.md`)
- [ ] Decision made: continue / expand / pause

---

*Haven — pilot checklist v1.0 — May 2025*
